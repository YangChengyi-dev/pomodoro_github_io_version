# 学习时间统计系统（GitHub.io 版本）

这是一个基于静态HTML和JavaScript的学习时间统计系统，可以托管在GitHub.io上。

## 功能特点

- 用户注册和登录
- 学习时间记录（开始/停止学习）
- 每日签到功能
- 学习数据统计和可视化图表
- 响应式设计，适配不同设备

## 技术栈

- HTML5
- CSS3 (Bootstrap 5)
- JavaScript
- Chart.js（用于数据可视化）

## 数据存储

所有数据都存储在浏览器的本地存储（localStorage）中，包括：
- 用户信息
- 学习记录
- 签到记录

## 如何部署到 GitHub.io

1. 在GitHub上创建一个新的仓库
2. 将 `github_io_version` 目录中的所有文件上传到仓库
3. 在仓库设置中启用 GitHub Pages
4. 选择 `main` 分支作为源
5. 访问生成的 GitHub.io 链接

## 使用方法

1. 访问部署后的网站
2. 注册一个新账号
3. 登录系统
4. 开始记录学习时间
5. 每日签到
6. 查看学习统计数据

## 注意事项

- 数据存储在本地浏览器中，清除浏览器数据会导致数据丢失
- 这是一个静态网站，不支持服务器端功能
- 仅用于个人学习和记录，不适合生产环境使用
